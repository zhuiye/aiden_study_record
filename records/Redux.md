### Redux

react 本身拥有管理state 的功能,也就是利用组件的生命周期函数,Redux 的定位是更好的**辅助管理**

### 什么情况下使用

1. react 解决不了的情况下再使用

2. 多个的组件需要用到全局的state
3. state 可预测,维护性,扩展性
4. 时间旅行,撤销更容易实现
