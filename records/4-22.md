双指针（对撞指针、快慢指针）

快慢 指针, 用于删除排序好的重复数组

如: [1,2,3,3,4,5]

i=0;
j=1;

```js
const removeDuplicates = (arr) => {
  let i = 0;
  let j = 1;
  for (let j = 1; j < arr.length; j++) {
    if (arr[j] !== arr[i]) {
      i++;
      arr[i] = arr[j];
    }
  }
  console.log(i + 1);
};
removeDuplicates([1, 1, 2, 2, 3, 3, 4]);
```
